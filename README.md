*MakeNote*

**个人记账与事务管理应用**
这是一个 Android 应用，旨在帮助用户进行日常记账、事务管理以及天气查询。应用结合了 SQLite 数据库存储、滑动页面切换、网络请求等功能，提供了简洁且高效的用户体验。

***主要功能***
****1. 用户注册与登录****
支持用户注册、登录，并使用 SQLite 存储用户信息。

****2. 界面与滑动页面****
应用使用 ViewPager 实现了流畅的页面滑动效果，用户可以通过滑动来切换不同的功能界面。

****3. 天气查询****
集成了 天气 API，可以获取用户当前地址的天气信息并显示在应用中。

****4. 记账功能****
用户可以记录每日的收入和支出，并通过 SQLite 存储。

支持新增、删除、修改收支记录，按日期查询收支情况。

****5. 事项管理****
用户可以添加、删除、修改和查询每日事务。

事务包含事项名称、开始和结束时间等信息，支持按年月日查询。
